Place report asset files here (served at /report-assets/).

Instructions:
- Add your organization logo as `logo.png` (recommended size ~100x100 or 300x300).
- The app will embed `/report-assets/logo.png` into downloaded PDFs automatically.

Example:
  staff-portal/public/report-assets/logo.png

If you prefer a different filename, update `logoUrl` in `src/pages/Reports.js` accordingly.
